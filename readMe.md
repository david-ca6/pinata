# pinata
Pinata is a simple pinata game where you throw Lemons at the Lumi pinata until it breaks, and candy falls out.
Type !throw in lumi's twitch chat to throw a lemon.

## library used
    - ebiten

## how to run
    On Linux & MacOS:
    ```bash
    ./run.sh
    ```
    On Windows:
    ```bash
      go run main.go
    ```
